# SMART Failure Prediction (HDD/SSD/NVMe) — ML Prototype

Прототип для преддипломной практики и ВКР по теме: **«Прогнозирование неисправности компьютерных компонентов методами машинного обучения»**.

Фокус проекта — **прогнозирование отказов накопителей (HDD/SSD/NVMe)** на основе диагностических **SMART/Health** показателей.

## Цель
Разработать и проверить подход к оценке **риска отказа диска** на заданном горизонте прогнозирования (например, *N дней*) по данным эксплуатации.

## Задачи
- собрать и описать источники SMART/Health данных;
- подготовить данные (очистка, обработка пропусков, формирование признаков);
- обучить базовые ML-модели и сравнить их качество;
- реализовать прототип расчёта риска отказа и подготовить материалы для отчёта.

## Источники данных
1) **Демонстрационные данные SMART/Health**, снятые в среде **RedOS 7**:
- NVMe SSD: `nvme smart-log`, `smartctl -a`
- SATA SSD / HDD: `smartctl -a`

2) **Публичный датасет SMART** (используется для обучения/оценки моделей), будет добавлен/указан в разделе `data/`.
